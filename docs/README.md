## 🌟 Introduction

### Vision
To become the global standard for affordable, all-in-one Enterprise Resource Planning, enabling businesses of all sizes to achieve maximum operational efficiency without the burden of subscribing to fragmented software solutions.

### Mission
Making Nepal and the global startup ecosystem digital by providing a unified, open-source-friendly, highly scalable, and developer-friendly operating system tailored for IT companies, agencies, and institutions.

### Project Description
Sovryx OS is an enterprise-grade Business Operating System. It centralizes every department into one platform including CRM, ERP, Accounting, Billing, Payroll, HRM, Project Management, Inventory, Sales, Marketing, Hosting Management, Support Desk, Knowledge Base, Training Institute, SaaS Management, and powerful AI tools.

## 🛠 Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5, AJAX, ApexCharts, DataTables, SweetAlert2.
- **Backend:** PHP 8.3+, MVC Custom Architecture, Composer.
- **Database:** MySQL 8+.
- **Security:** PHP Sessions, RBAC, CSRF Protection, PDO, Password Hashing.
- **Integrations:** PHPMailer, WhatsApp Business API, Dompdf, OpenAI API.
