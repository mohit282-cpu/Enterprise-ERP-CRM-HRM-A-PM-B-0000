# User Guide

This guide is intended for Managers, Accountants, Sales Staff, and Project Managers using **Sovryx OS** for their daily operations.

---

## 📈 1. CRM & Sales Workflow

### Managing Leads
1. Navigate to **CRM > Leads**.
2. Click **Add Lead** to manually enter a potential client, or import them via CSV.
3. Move leads through the Kanban board stages (e.g., *New*, *Contacted*, *Qualified*, *Lost*).
4. Use the **AI Email Assistant** inside the lead profile to instantly draft follow-up emails.

### Converting Leads to Clients
Once a lead agrees to do business, click **Convert to Client**. This moves the record to the Client Database and unlocks Project and Billing features for that entity.

---

## 💼 2. Projects & Task Management

### Creating a Project
1. Go to **Projects > All Projects** and click **New Project**.
2. Select the associated Client.
3. Define the Scope, Start Date, and Deadline.
4. Assign team members (Developers, Designers, etc.).

### Managing Tasks
- Inside a project, navigate to the **Tasks** tab.
- Add actionable tasks, assign them to team members, and set priorities.
- Employees can click **Start Timer** on a task to begin logging billable hours.

---

## 💰 3. Billing and Accounting

### Generating a Quotation / Proposal
1. Go to **Billing > Quotations**.
2. Click **Create Quotation**. Select the Client.
3. You can manually add line items or click **Generate with AI** and type a prompt (e.g., *"Write a proposal for a 5-page WordPress website"*).
4. Click **Send to Client**. They will receive an email with a secure link to accept or decline.

### Creating an Invoice
- If a quotation is accepted, click **Convert to Invoice**.
- To create a manual invoice, go to **Billing > Invoices > Create**.
- Fill out the line items, taxes, and discounts.
- Click **Generate PDF** to download it, or **Email Invoice** to send it directly.

### Accounting (Revenue & Expenses)
- When a client pays an invoice, click **Add Payment**. This automatically credits the Revenue account.
- To log a company expense (e.g., Office Rent, Server Costs), go to **Accounting > Expenses > Add Expense**. Select the appropriate Chart of Account category.
- View the **Profit & Loss Statement** in the Reports module to see real-time financial health.

---

## 🌐 4. IT Operations (Hosting & Domains)

*For web agencies and IT companies.*

### Domain Tracking
1. Go to **IT Ops > Domains**.
2. Add a client's domain, the registrar, and the **Expiry Date**.
3. The system will automatically send reminder emails/WhatsApp messages 30, 15, and 3 days before expiry.

### Server & Hosting Assets
- Track physical servers, VPS instances, or shared hosting packages.
- Link them to specific clients for easy billing generation upon renewal.

---

## 🤖 5. Using the AI Assistant

Look for the **✨ AI Assist** button throughout the system (CRM, Support Tickets, Proposals).
- **In Support Tickets:** The AI can read a long client email and generate a 2-sentence summary.
- **In Emails:** Type "Tell the client their project is delayed by 2 days due to server issues" and the AI will draft a professional, polite email.
